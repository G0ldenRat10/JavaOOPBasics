# JavaOOPBasics

Ovaj repozitorij sadrži vežbe i primere iz kursa Java OOP Basics.

Sadržaj:
- Razne Java klase i primeri iz predavanja
- Struktura projekta je u `src/` direktorijumu

Kako pokrenuti:
1. Otvori projekat u IntelliJ IDEA
2. Kompajliraj i pokreni željeni `Main.java` fajl iz odgovarajuće podmape u `src/`

Live templates korisni u ovom projektu:
- `sysout` — proširuje se na `System.out.println()` (ako je importovan iz .idea/liveTemplates)
